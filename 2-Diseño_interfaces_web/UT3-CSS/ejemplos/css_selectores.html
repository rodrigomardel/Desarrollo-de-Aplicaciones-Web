Introducción a los Selectores CSS

Los selectores CSS permiten aplicar estilos a los elementos HTML de manera precisa. Existen varios tipos de selectores que nos permiten ser muy específicos o generales a la hora de definir los estilos. Los selectores más básicos incluyen tipos de elementos, clases y IDs.

Selector de Clase y ID

El selector de clase se utiliza con el símbolo . seguido del nombre de la clase, y se puede aplicar a varios elementos. El selector de ID utiliza el símbolo # y es único por página.

Recuerda que el uso correcto de selectores optimiza el rendimiento de la página.

Selector de Pseudo-clase

Las pseudo-clases se utilizan para definir un estado especial de un elemento, como cuando el ratón pasa por encima de un enlace (:hover).

En este ejemplo, hemos aplicado una pseudo-clase :hover a los enlaces para cambiar su color cuando se interactúa con ellos.

Selector de Atributo

El selector de atributo se utiliza para seleccionar elementos en base a un atributo y su valor. Por ejemplo, podemos estilizar todos los enlaces que abren en una nueva pestaña utilizando a[target="_blank"].
